Publishing rule

The GUI publishes feed files into web/feeds. The publish helper archives old feed variants from outputs/feeds, docs/feeds, and non-current files in web/feeds into each folder's local .archived folder. Archived files are ignored by Git.

TiviMate URLs

Use the GitHub Pages URLs. Do not use tokenized raw GitHub URLs.

Security: Feed files can contain provider stream URLs. Keep repository and GitHub Pages exposure aligned to intended access.
