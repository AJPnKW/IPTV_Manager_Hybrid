IPTV Output Feeds

This folder is the canonical output location for generated focused IPTV feed files.

The GUI publish action replaces this page with a generated feed index after the next parse/publish run.

Expected stable files:

  • latest_focused_playlist.m3u8
  • latest_focused_epg.xml
  • latest_scoped_playlist.m3u8
  • latest_scoped_epg.xml
  • feed_manifest.json
